### 浮点数二分一般用于解方程

题目：求数的三次方根

```c++
bool check(double x)
{
    if (x * x * x <= n)
        return true;
    return false;
}

// 这里用的是左边为蓝色的思想
double Fbinary_search()
{
    double l = -100, r = 100; // 根的范围
    // 这里放弃用while来判断左右边界，因为误差很大
    // 直接让它二分100次
    for (int i = 0; i < 100; i++)
    {
        double mid = (l + r) / 2;
        if (check(mid))
            l = mid;
        else
            r = mid;
    }
    // 二分了100次，l和r基本相同，随便返回一个就好
    return l;
}
```
