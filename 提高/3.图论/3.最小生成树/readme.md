##### prim

prim算法：维护一棵树，每次从外界选择一条权值最小的边加入树中，保证加边后仍然为树，直到树包含所有顶点为止

时间复杂度：$O(n^2)$，用邻接矩阵存图

证明：当前与外界直接相连的权值最小的一条边ab，这条边一定可以出现在最优解中；此时a在树中，b在树外

反证：此时ab不连通，由于所有点都会被加入到mst，若不选该边，后续ab会以其他路径x连通
     视边ab与“其他路径x”一起形成环，x上有若干边，其中至少有一条边y权值>=ab的权值，因为它的出现和选择在ab之后，ab先于y被选中，说明ab边权更小
     由于是无向边，因此y若用ab来替代，则会得到更好的结果；因此矛盾

##### kruskal

kruskal算法：每次从小到大选择一条边加入树中，保证加边后仍然为树，直到树包含所有顶点为止

时间复杂度：$O(m\log m)$，用三元组存边

证明：当前权值最小的一条边ab，这条边一定可以出现在最优解中

反证：思路类似同上：此时ab不连通，但ab总会连通；如果不选该边，后续ab会以其他路径x连通
     视边ab与“其他路径x”一起形成环，x上有若干边，其中至少有一条边y权值>=ab的权值，因为它的出现和选择在ab之后，排序说明y边权更大
     由于是无向边，因此y若用ab来替代，则会得到更好的结果；因此矛盾

##### 思想应用
如何证明当前边a一定可以选？

反证：假设不选当前边，得到了一棵树；将a加上后，那么必然出现环；在这个环上，一定可以找到一条长度不小于a的边，那么用a来替换这条边，结果一定不会变差
