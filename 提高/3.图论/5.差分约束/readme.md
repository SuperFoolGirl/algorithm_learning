### 作用
1. 求不等式组的可行解
2. 求不等式组的最优解(最大值/最小值)



### 不等式组，形如
$x_i \leq x_j + c_k$

其中，$x_i$和$x_j$是变量，$c_k$是常数

不等式组由若干个上述方程构成，符号必须统一，不得混用$\leq$和$\geq$



### 如何求解
$x_i \leq x_j + c_k$

观察形式，很像图论中的**三角不等式**(松弛方程)

不过，求最短路时，我们用的是dist[j]>dist[t]+w，看起来与这个相反

但是，如果从某个起点出发，求完所有最短路后，再去遍历所有点，那么dist[j]<=dist[t]+w将恒成立

也就对应上了上述不等式组的形式

因此，将不等式组里的每个不等式视为一条边，$x_i$和$x_j$视为边的两个端点，$c_k$视为边的权重

显然，如果原不等式组形如$x_i \geq x_j + c_k$，那么用最长路算法求解即可(与求最小值有关，下面会提到)



### 解的判断
#### 可行解
源点需要满足的条件：从源点出发，一定可以走到所有的边。注意，只要存在一个满足该条件的源点，那么整个不等式组就有可行解了，而不是只关乎源点对应变量的解

边对应的是不等式，只有走过所有边，才能保证所有不等式都满足；如果有边无法走到，说明有不等式无法满足，相当于原不等式组漏了一些不等式

额外考虑下，如果存在走不到的点i，意味着什么？

从图论角度，由于这个模型是有向图，因此不能判断其连通性；从不等式组角度，说明变量i与方程组的解无关，其任意取值都满足不等式组

步骤如下：

1. 将不等式$x_i \leq x_j + c_k$转化为一条从$x_j$到$x_i$的边，权重为$c_k$
2. 找一个超级源点，使得该源点一定可以遍历到所有边
3. 对该源点做最短路。若存在负环，则无可行解；否则，dist数组即为一个可行解



#### 最优解（最大值/最小值，指每个变量的最值）
首先解释一下“每个变量的最值”

对于原不等式组，应当能写成一个不等式链，即确定出所有变量的相对大小关系

让所有变量都取到最大值/最小值，依然满足相对关系的约束，就是我们要寻求的最优解

不过，既然已经说了是“相对关系”了，若要求出具体最值，应该得到绝对关系才是——也就是，不等式链的两端，应该给出上界/下界

因此，求最值问题，一般会给出限制，比如$x_i \leq c$，或者$x_i \geq c$，或者$x_i$的值必须在某个范围内

**结论：如果求最小值，则应该求最长路；如果求最大值，则应该求最短路**

另外，根据上面所讲内容，再结合下面讲解的一点剧透，需要声明：最大值和最小值，在同一个不等式组中，只能求其中一个

**如果不等式组全都是<=，则只能求最大值；如果不等式组全都是>=，则只能求最小值；同时，这也规定了不等式组的构成——符号统一**

$x_i \leq x_j + c_k$用于构建不等式链上界(上界就是最大值)，求最小上界才能满足交集，故求最短路

$x_i \geq x_j + c_k$用于构建不等式链下界(下界就是最小值)，求最大下界才能满足交集，故求最长路

如果题目要求同时求出最小值和最大值，那么我们需要建立两个不等式组，在两个图上分别求最短路/最长路



##### 考虑处理$x_i \leq c$
方法：建立超级源点0，然后对所有xi建立0->xi，权重为c



##### 以求xi最大值为例
首先，根据高中知识，若要求x的最大值，应当有形如`x<=c`这种形式，即“x的上限是c，但下限未知”

所有**从xi出发**的不等式链`xi<=xj+c1<=xk+c1+c2<=...<=c1+c2+...+cn`所计算出的上界，最终xi的最大值等于所有上界的最小值

以`xi<=2, xi<=3, xi<=5`为例，为了使整个不等式组成立，需要求解的交集，也就是“所有上界的最小值”

然后，回到不等式链本身，还有一些有趣的性质

这个不等式链，还原到图中是怎样的呢？给出结论：`0->...->xk->xj->xi`，其中0为超级源点

而且，整条路径的权重和为`c1+c2+...+cn`，也就是xi的上界

因此，每一个不等式链，都对应一个从0到i的路径；不等式的上界即为路径长度

因此，**要求上界最小值，等价于求所有从0到i的路径的最小值，也就是最短路。而“上界”本身这个概念就是“最大值”**



##### 类似地，考虑求xi最小值
首先，题目在不等式组中应给出`xi>=c`的形式，即“xi的下限是c，但上限未知”

从xi出发，构建不等式链`xi>=xj+c1>=xk+c1+c2>=...>=c1+c2+...+cn`，最终xi的最小值等于所有下界的最大值(注意，这里要求">="的不等式)

对应路径仍然为`0->...->xk->xj->xi`，路径权重和为c1+c2+...+cn，即xi的下界

因此，**要求下界最大值，等价于求所有从0到i的路径的最大值，也就是最长路**
