### 二分图 $\Leftrightarrow$ 不存在奇数环 $\Leftrightarrow$ 染色法不存在矛盾
**首先提一下，二分图是无向图**

这里给出粗略证明：

1. 不存在奇数环 $\Leftrightarrow$ 染色法不存在矛盾：取连通块上一奇数长度路径，则一定有首尾点颜色相同；连接首尾点形成奇数环，此时染色法就会出现矛盾
2. 二分图 $\Leftrightarrow$ 染色法不存在矛盾：成功染色的状态下，所有边的两端必定是一黑一白；此时把白色点全拿出来放到集合1，黑色点全拿出来放到集合2，此时两集合内部没有边，满足二分图定义
3. 由于b/c等价，a/c等价，因此a/b等价，进而有a/b/c等价
