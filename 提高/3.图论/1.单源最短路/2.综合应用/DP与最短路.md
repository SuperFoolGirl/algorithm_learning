以01背包问题`f[i][j] = max(f[i-1][j], f[i-1][j-v]+w)`为例

1. 我之前描述DP为“被扯着向前”在图中行进，枚举当前点是由哪个前驱扩展而来；如果说`f[n][m]`是终点，那么起点其实应该是若干个：`f[0][0], f[0][1],..., f[0][m]`
2. 将`f[i][j]`视为一个点，则它可以由两个点扩展而来：`f[i-1][j]`边权为`0`，`f[i-1][j-v]`边权为`w`；整个图共有`(n+1)×(m+1)`个点
3. 状态转移方程为max属性，则对应求**最长路**
4. 由于DP描绘的图一般无环，因此**大部分DP是拓扑图上的最短路问题**；如果是拓扑图，那么就可以按照拓扑序列线性地求出最短路（参考3.道路与航线），而不用复杂度更高的最短路算法。线性复杂度：拓扑排序为`O(V+E)`，DP为`O(n*m*k)`，其中k为决策，也就是“前驱数量”，同样都是“点+边”
5. 反过来，对于非拓扑序的DP问题，可以将其转为用dijkstra、spfa求解的最短路问题
