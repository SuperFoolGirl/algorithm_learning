### Floyd算法应用
1. 最短路
2. 传递闭包
3. 找最小环
4. 恰好经过k条边的最短路(倍增思想)



### Floyd算法原理
> Floyd算法是基于DP的

初始化：

d[i][j] = +∞, i != j
d[i][j] = 0 , i == j

状态表示 f(k,i,j)

集合：所有从i出发，最终走到j，且中间只经过节点编号不超过k的路径

属性：min

状态计算：

找最后不同的一步来分类：k是否在路径上

1. k不在路径上：f(k,i,j) = f(k-1,i,j)
2. k在路径上：f(k,i,j) = f(k-1,i,k) + f(k-1,k,j)
   注意，i~k与k~j是无关的两条路径，可以直接相加；求和的最小值，即求最小值的和

由状态转移方程，只用到k-1层，因此可以使用滚动数组
