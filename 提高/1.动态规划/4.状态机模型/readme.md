首先，什么是状态机？

回到时序逻辑电路的状态图，不论是同步还是异步，展现的就是状态机

给定一个状态和一个输入，总能确定地转换到下一个状态



对于之前见过的DP问题，这里以01背包为例

在状态转移方程中，表述出了两种情况：选i或不选i

这两个情况都只有1步——选，就结束了；不选，也结束了——没有中间过程

但有些题目，中间的过程可能很复杂，甚至有很多步，这时候就需要状态机了



状态机题目的特征：描述的是一个过程，而非结果

选或不选，是很直接的结果；在状态机里，相当于是把这个“直接的结果”扩展成过程

因此，可以将状态机问题归类为“另类的状态表示方式”

另类的状态表示分为两类：一种是状态机，另一种是状态压缩

有些时候，只用一个数字难以表示一个状态，因此就用状态压缩/状态机的方式来表示状态
