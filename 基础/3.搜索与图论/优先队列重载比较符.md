默认大根堆，最大值优先

对于结点是结构体的情况，需要自定义比较函数

有两种方式：
1. 自定义比较类，重载()运算符
2. 重载结构体的<运算符

推荐第二种，更简洁。

小根堆同理，重载>运算符

记忆技巧：
1. 反着记就行了，大根堆排序用小于，小根堆排序用大于（）
2. 不论哪种重载，比较符的左值都是this指针指向的对象，也就是“待比较的元素”

大根堆重载<
```cpp
struct Node {
    int id;
    int val;
    
    // 重载<运算符，实现大根堆
    bool operator<(const Node& other) const {
        return val < other.val;  // val大的优先级高
    }
};
priority_queue<Node> maxHeap;  // 大根堆
```

小根堆重载>
```cpp
struct Node {
    int id;
    int val;
    
    // 重载>运算符，实现小根堆
    bool operator>(const Node& other) const {
        return val > other.val;  // val小的优先级高
    }
};
priority_queue<Node, vector<Node>, greater<Node>> minHeap;  // 小根堆
```
